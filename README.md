# Portfolio Professionnel

> Site web vitrine présentant mes projets, mes compétences et mon parcours de développement.

---

## Présentation du Projet

Ce dépôt contient le code source de mon portfolio en ligne. L'objectif de ce site est de centraliser mes réalisations techniques, de décrire mes expertises sectorielles et de fournir un canal de communication direct pour d'éventuelles collaborations professionnelles.

L'accent a été mis sur la performance, l'accessibilité et une expérience utilisateur fluide, notamment grâce à une approche **Responsive Design** adaptée à l'ensemble des résolutions d'écrans.

---

## Fonctionnalités Principales

| Section | Description | Fonctionnalité clé |
| :--- | :--- | :--- |
| **Accueil** | Introduction et proposition de valeur. | Accès rapide au CV et résumé du profil. |
| **À Propos** | Détail du parcours et de la philosophie de travail. | Chronologie interactive des expériences. |
| **Projets** | Galerie de mes réalisations techniques. | Système de filtrage par technologies/mots-clés. |
| **Compétences** | Visualisation de mes expertises techniques. | Catégorisation par niveau de maîtrise. |
| **Contact** | Formulaire de mise en relation directe. | Validation des champs côté client et sécurisation. |

---

## Technologies Utilisées

* **Front-end :** HTML5, CSS3 (architecture en variables et Flexbox/Grid), JavaScript.
* **Hébergement :** Déploiement continu configuré via GitHub Actions.
